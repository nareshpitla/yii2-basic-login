# yii2-basic-login

 yii2 Framework basic module integrated with signup and login module system along with email verification.
 
 Framework is pre installed with bootstrap 4 version and MDB 
 
 after download change db parameters as per your db credentials in config/db.php
 
 to use email verification on signup, change parameters in mailer component in config/web.php
 
 Before use, Run yii migration command php yii migrate to create user and migration tables in database for login and signup purpose.
